# opencv-coral-4.1.1

## General Informations
Compilation Opencv for coral 4.1.1 + dependencies

## As You Need
You need only to clone the repository and launch the shell script

## Setup
```
$cd opencv-coral-4.1.1
$chmod 755 init.sh
$bash init.sh
```
